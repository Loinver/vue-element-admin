# 打包发布

## Project setup 项目安装依赖
```
yarn install
```

### Compiles and hot-reloads for development  开发环境命令
```
yarn serve
```

### Compiles and minifies for production  生产环境命令
```
yarn build:uat  (测试环境)
yarn build:prod (生产环境)
```

### Lints files  格式化文件，查看语法错误
```
yarn lint
```

### fixes files  修复文件语法错误
```
yarn fix
```
