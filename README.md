# 打包发布

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build:uat  (测试环境)
yarn run build:prod (生产环境)
```

### Lints and fixes files
```
yarn run lint
```
